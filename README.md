Setup
----------------

1. install [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh)

2. cd your-dir

3. `git clone git://github.com/yoshiori/oh-my-zsh-yoshiori.git`

4. `cd oh-my-zsh-yoshiori`

5. `sh linkmaker.sh`

6. `source ~/.zshrc`
